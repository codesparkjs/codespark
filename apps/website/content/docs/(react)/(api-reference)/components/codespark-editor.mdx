---
title: CodesparkEditor
description: A flexible code editor component supporting Monaco Editor and CodeMirror.
---

`CodesparkEditor` is a standalone code editor component that supports both Monaco Editor and CodeMirror as the underlying editor engine. It integrates with the Workspace for file management and automatic compilation.

```tsx
import { CodesparkEditor } from '@codespark/react';
```

## Basic Usage

```tsx
<CodesparkEditor value="some code..." />
```

```tsx codespark-preview height=249
import { CodesparkEditor } from '@codespark/react';

const code = `import { CodesparkEditor } from '@codespark/react';

export default function App() {
  return <CodesparkEditor />;
}`;

export default function App() {
  return <CodesparkEditor value={code} />;
}
```

### toolbox

Toolbar configuration. Set to `false` to hide, `true` to show default tools, or pass a custom array.

| Type | Default | Since |
| ---- | ------- | ----- |
| `boolean \| (ToolboxItemId \| ToolboxItemConfig \| ReactElement)[]` | `['reset', 'format', 'copy']` | v1.0.0 |

Built-in tool IDs:
- `'reset'` - Reset code to initial state
- `'format'` - Format the code
- `'copy'` - Copy code to clipboard

```tsx twoslash preview="codespark-editor-toolbox"
// @noErrors
import { CodesparkEditor } from '@codespark/react';
// Hide toolbar
<CodesparkEditor toolbox={false} />

// Custom toolbar
<CodesparkEditor toolbox={['copy', 'format']} />

// Custom tool item
<CodesparkEditor
  toolbox={[
    'copy',
    {
      tooltip: 'My Custom Action',
      icon: <MyIcon />,
      onClick: editor => console.log(editor?.instance)
    }
  ]}
/>
```

## Monaco-specific Props

These props only apply when using Monaco Editor:

### options

Monaco editor options.

| Type | Default | Since |
| ---- | ------- | ----- |
| `IStandaloneEditorConstructionOptions` | - | v1.0.0 |

```tsx twoslash
// @noErrors
import { CodesparkEditor } from '@codespark/react';
import { Monaco } from '@codespark/react/Monaco';

<CodesparkEditor
  editor={Monaco}
  options={{
    readOnly: true,
    minimap: { enabled: true },
    folding: true,
    lineNumbers: 'off'
  }}
/>
```

```tsx codespark-preview height=249
import { CodesparkEditor } from '@codespark/react';
import { Monaco } from '@codespark/react/monaco';

const code = `export default function App() {
  return (
    <div className="text-center">
      <h1 className="text-xl font-bold">Hello World</h1>
      <p>This is my first Codes Park!</p>
    </div>
  );
}`;

export default function App() {
  return (
    <CodesparkEditor
      editor={Monaco}
      value={code}
      toolbox={[]}
      options={{
        readOnly: true,
        minimap: { enabled: true },
        folding: true,
        lineNumbers: 'off'
      }}
    />
  );
}
```
