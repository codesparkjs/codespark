---
title: Link
description: Injects external resources into the preview iframe.
---

`Link` is an injection component that adds external resources to the preview iframe. It renders nothing in the React tree but injects a `<link>` tag into the iframe's document. Commonly used for external stylesheets, fonts, or other linked resources.

```tsx
import { Link } from '@codespark/react';
```

## Basic Usage

Pass the `Link` component as a child of `Codespark` or `CodesparkPreview`:

```tsx twoslash preview
// @noErrors
import { Codespark, Link, Style } from '@codespark/react';

const code = `export default function App() {
  return (
    <div className="font-dancing text-3xl text-center p-4">
      Beautiful Custom Font
    </div>
  );
}`;

export default function App() {
  return (
    <Codespark code={code}>
      <Link
        rel="stylesheet"
        href="https://fonts.googleapis.com/css2?family=Dancing+Script:wght@700&display=swap"
      />
      <Style>{`.font-dancing { font-family: 'Dancing Script', cursive; }`}</Style>
    </Codespark>
  );
}
```

## Props

`Link` accepts all standard HTML `<link>` attributes:

### rel

Relationship between the current document and the linked resource.

| Type | Default | Since |
| ---- | ------- | ----- |
| `string` | - | v1.0.0 |

Common values:
- `'stylesheet'` - External CSS file
- `'preconnect'` - Preconnect to a server
- `'prefetch'` - Prefetch a resource
- `'icon'` - Favicon

```tsx twoslash
// @noErrors
import { Link } from '@codespark/react';
// ---cut---
<Link rel="stylesheet" href="https://example.com/styles.css" />
```

### href

URL of the linked resource.

| Type | Default | Since |
| ---- | ------- | ----- |
| `string` | - | v1.0.0 |

```tsx twoslash
// @noErrors
import { Link } from '@codespark/react';
// ---cut---
<Link rel="stylesheet" href="https://example.com/styles.css" />
```

### type

MIME type of the linked resource.

| Type | Default | Since |
| ---- | ------- | ----- |
| `string` | - | v1.0.0 |

```tsx twoslash
// @noErrors
import { Link } from '@codespark/react';
// ---cut---
<Link rel="stylesheet" type="text/css" href="https://example.com/styles.css" />
```

### media

Media query for the linked resource.

| Type | Default | Since |
| ---- | ------- | ----- |
| `string` | - | v1.0.0 |

```tsx twoslash
// @noErrors
import { Link } from '@codespark/react';
// ---cut---
<Link
  rel="stylesheet"
  href="https://example.com/print.css"
  media="print"
/>
```

### crossOrigin

CORS setting for the request.

| Type | Default | Since |
| ---- | ------- | ----- |
| `'anonymous' \| 'use-credentials'` | - | v1.0.0 |

```tsx twoslash
// @noErrors
import { Link } from '@codespark/react';
// ---cut---
<Link
  rel="stylesheet"
  href="https://example.com/styles.css"
  crossOrigin="anonymous"
/>
```

### Other Attributes

Any other valid `<link>` HTML attributes are also supported.

## Use Cases

### External Stylesheets

```tsx twoslash
// @noErrors
import { Codespark, Link } from '@codespark/react';
// ---cut---
<Codespark>
  <Link rel="stylesheet" href="https://cdn.example.com/library.css" />
</Codespark>
```

### Google Fonts

```tsx twoslash
// @noErrors
import { Codespark, Link } from '@codespark/react';
// ---cut---
<Codespark>
  <Link rel="preconnect" href="https://fonts.googleapis.com" />
  <Link rel="preconnect" href="https://fonts.gstatic.com" crossOrigin="anonymous" />
  <Link
    rel="stylesheet"
    href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
  />
</Codespark>
```

### Icon Libraries

```tsx twoslash
// @noErrors
import { Codespark, Link } from '@codespark/react';
// ---cut---
<Codespark>
  <Link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css"
  />
</Codespark>
```

### CSS Frameworks

```tsx twoslash
// @noErrors
import { Codespark, Link } from '@codespark/react';
// ---cut---
<Codespark>
  <Link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
  />
</Codespark>
```

### Preconnect for Performance

```tsx twoslash
// @noErrors
import { Codespark, Link } from '@codespark/react';
// ---cut---
<Codespark>
  <Link rel="preconnect" href="https://api.example.com" />
  <Link rel="dns-prefetch" href="https://cdn.example.com" />
</Codespark>
```

## Complete Font Example

Here's a complete example loading a custom Google Font:

```tsx twoslash preview
// @noErrors
import { Codespark, Link, Style } from '@codespark/react';

const code = `export default function App() {
  return (
    <div className="space-y-4 p-4">
      <h1 className="font-playfair text-3xl font-bold">
        Elegant Typography
      </h1>
      <p className="font-inter text-gray-600">
        Mix different fonts for headings and body text.
      </p>
    </div>
  );
}`;

const fontStyles = `
.font-playfair { font-family: 'Playfair Display', serif; }
.font-inter { font-family: 'Inter', sans-serif; }
`;

export default function App() {
  return (
    <Codespark code={code}>
      <Link rel="preconnect" href="https://fonts.googleapis.com" />
      <Link rel="preconnect" href="https://fonts.gstatic.com" crossOrigin="anonymous" />
      <Link
        rel="stylesheet"
        href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500&family=Playfair+Display:wght@700&display=swap"
      />
      <Style>{fontStyles}</Style>
    </Codespark>
  );
}
```

## Loading Order

Links are injected into the iframe's `<head>` element before the preview renders. External stylesheets load asynchronously, so there may be a brief flash of unstyled content (FOUC) for large stylesheets.

<Callout title="Performance Tip">
Use `rel="preconnect"` for domains you'll load resources from to improve loading performance.
</Callout>
