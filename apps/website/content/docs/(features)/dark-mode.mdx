---
title: Dark Mode
description: Enable dark mode to match your design system.
---

Codespark has a fluent dark mode that matches your design system perfectly. Use the `theme` prop to enable dark mode. Click the theme toggle button on the right side of the page header to preview the dark mode effect.

```tsx twoslash
import { Codespark } from '@codespark/react';
// ---cut---
// @noErrors
<Codespark theme="" />
//                ^|
```

```tsx codespark
import { useState } from 'react';

export default function App() {
  const [count, setCount] = useState(0);

  return (
    <button className="rounded-lg bg-black px-4 py-2 text-white dark:bg-white dark:text-black" onClick={() => setCount(c => c + 1)}>
      Count: {count}
    </button>
  );
}
```
