title: Markdown

---

In addition to React syntax, Codespark also supports markdown rendering. There are two ways to preview markdown code:

- Import markdown file as a React component
- Use framework for pure markdown rendering

## Import Markdown File

You can import a markdown file just like a regular React component.

```ts
import Doc from './doc.md';

<Doc />
```

```tsx codespark-preview
import { Codespark } from '@codespark/react';

const files = {
  './App.tsx': `import Doc from './doc.md';

export default function App() {
  return <Doc />
};
`,
  './doc.md': '# Hello World'
}

export default function App() {
  return <Codespark files={files} tailwindcss={false} />;
}
```

## Use Markdown Framework

If you choose to use the Markdown Framework, your `<Codespark />` component will only support Markdown syntax.

First, you need to install an additional npm package, then pass the `framework` prop to the component.

```npm
npm install @codespark/framework
```

```tsx twoslash
import { markdown } from '@codespark/framework/markdown';

// @noErrors
<Codespark framework={markdown} />;
```

Once the framework is configured, simply pass your markdown code to the `code` prop to preview it.

```tsx preview
import { markdown } from '@codespark/framework/markdown';
import { Codespark } from '@codespark/react';

export default function App() {
  return <Codespark framework={markdown} name="index.md" code="# Hello World" tailwindcss={false} />;
}
```

<Callout title="Note">
By changing the name suffix to .md like `index.md` will enable markdown syntax highlighting in the editor.

And you also need to disable Tailwind CSS by setting `tailwindcss={false}`, because Tailwind CSS normalizes default element styles with [Preflight](https://tailwindcss.com/docs/preflight).
</Callout>
