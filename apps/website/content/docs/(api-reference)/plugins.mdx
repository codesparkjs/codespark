Codespark provides plugins for popular build tools to integrate live code demos into your documentation.

<Cards>
  <Card href="/packages/plugin-rollup" title="Rollup">
    Transforms JSX/TSX code blocks in Rollup builds with configurable methods.
  </Card>
  <Card href="/packages/plugin-remark" title="Remark">
    Converts markdown code blocks with codespark directives into live demo components.
  </Card>
</Cards>
